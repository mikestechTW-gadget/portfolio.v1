Case Study: Full IT & ERP Modernization (2024)

Manufacturing factory • Progressive rollout across 2024

Context

  • Role: Outsourced IT vendor (Owner)
  • Scale: ~65 employees
  • Wi-Fi coverage: ~2000 m²
  • Timeline: Progressive modernization across 2024
Goal: Remove instability and legacy bottlenecks while improving security and operational efficiency.

Tech & components

  • Wired network restructure (switch replacement)
  • Wi-Fi roaming upgrade using Aruba
  • Firewall upgrade to FortiGate UTM + custom policy design
  • Server replacement + ERP modernization with Ragic
  • Endpoint health check and phased PC refresh

Problem

  • Unstable wired and wireless network environment
  • Aging PCs leading to low productivity
  • Legacy firewall with insufficient protection
  • Old server and outdated ERP slowing operations

Approach

  • Company-wide network assessment and architecture redesign
  • Switch replacement and Wi-Fi roaming upgrade (Aruba)
  • PC health checks and replacement roadmap
  • FortiGate UTM migration with customized policies
  • Server replacement and Ragic ERP implementation

Results

Reliability
Network instability issues eliminated
Security
Risk reduced through policy hardening
Business
30%+ efficiency • Overtime reduced • 15% labor cost saved
Takeaway: Combining infrastructure stability, security policy hardening, and ERP modernization created direct operational impact—not only fewer IT issues, but measurable productivity gains.
